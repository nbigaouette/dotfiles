# dotfiles

## Install

```sh
git clone git@github.com:nbigaouette/dotfiles.git ~/.dotfiles
~/.dotfiles/install.sh
```

## What is included

* Tmux configuration, including TPM and some plugins;
* Neovim configuration, usable with vim
